Алгоритм карьеры в IT

  1. Закончить курс JavaScript.
  2. Сделать несколько проектов для своего портфолио с использованием новых полученных знаний.
  3. Найти подработку в качестве верстальщика.
  4. Совершенстовать полученные знания.
  5. Читать дополнительную литературу.
  6. Работать над новыми проектами, применяя более сложные знания и умения.
  7. Начать курс по изучению PHP.
  8. Параллельно изучению PHP заниматься разработкой проектов.
  9. Освоить несколько популярных CMS.
  10. Изучить возможности Wordpress.
  11. Создать несколько готовых проектов на Wordpress.
  12. Изучить возможности Joomla.
  13. Создать несколько проектов на Joomla.
  14. Изучить возможности Drupal.
  15. Создать несколько проектов на Drupal.
  16. Создать готовый проект, используя язык PHP.
  17. Изучить особенности работы на Python.
  18. Изучить особенности работы на C#.
  19. Изучить особенности работы на Ruby.
  20. Найти свою нишу.
  21. Устроиться на работу в компанию в качестве веб-разработчика.
